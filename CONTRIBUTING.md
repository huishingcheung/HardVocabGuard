# 贡献指南（CONTRIBUTING）

感谢你愿意贡献本项目。

## 开发规范

- 遵循 Android 官方规范与 Kotlin 官方代码风格
- 禁止引入广告/统计/埋点 SDK
- 禁止引入非必要第三方依赖，优先使用 Jetpack 官方组件
- 所有新增代码必须包含清晰的中文注释与边界说明

## 提交要求

- 提交信息建议格式：`type: 简短描述`
  - `feat:` 新功能
  - `fix:` 修复问题
  - `refactor:` 重构
  - `docs:` 文档
  - `test:` 测试

## 兼容性要求

- `minSdkVersion 34` 不降低
- `targetSdkVersion 36` 不降低
- 不使用废弃 API；如确需兼容差异，必须说明原因与替代方案

## 安全与隐私

- 所有用户数据必须仅本地存储
- 禁止任何形式的网络上传/收集
